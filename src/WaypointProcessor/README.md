# WaypointProcessor

# How To

## Build
```
dotnet.exe build .\WaypointProcessor.csproj
```

## Run
```
dotnet run .\bin\Debug\net8.0\WaypointProcessor.dll
```

## Usage
```
Required option 'b, baseFileName' is missing.
  Required option 'c, comparedFileName' is missing.

  -b, --baseFileName        Required. Base file name

  -c, --comparedFileName    Required. Compared file name

  -d, --distance            Max distance to consider waypoints equal

  --help                    Display this help screen.

  --version                 Display version information.
```
